### 古典密码学

> 起源：古代战争--古典密码学

* 移位式加密

  ![移位加密](https://dingyx.oss-cn-shenzhen.aliyuncs.com/picgo/移位加密.png)

  ```
  china->FKLQD
  
  c->F
  h->K
  i->L
  n->Q
  a->D
  ```

  * 加密算法：位移
  * 密钥：位移数

* 替换式加密

  ![替换加密](https://dingyx.oss-cn-shenzhen.aliyuncs.com/picgo/替换加密.png)

  ```
  china->HTJNW
  
  c->H
  h->T
  i->J
  n->N
  a->W
  ```

  * 加密算法：替换文字
  * 密钥：码表
